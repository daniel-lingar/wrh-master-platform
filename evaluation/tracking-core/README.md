# WRH Pilot Operating & Evaluation Package

**Program Administrator:** Capitol Contracts LLC  
**Program:** What Really Happened™ (WRH)  
**Package Type:** Pilot-operating infrastructure and evaluation system  

---

## Purpose

This repository contains the operating documents used to run and evaluate a controlled WRH pilot.

The package answers one question:

> **Can WRH be run safely once, measured honestly, and improved before it scales?**

This repository is not the curriculum source. It is not the sales packet. It is not a clinical treatment file. It is the pilot operations and evidence layer.

---

## Scope Boundary

WRH is a non-clinical psychoeducational curriculum. This pilot package measures operational performance only.

It evaluates:

- curriculum delivery
- facilitator fidelity
- attendance and completion
- participant feedback
- activation events
- Closed-Loop Support Bridge performance
- unresolved handoffs
- site performance
- required revisions before continuation or expansion

It does **not** claim or measure:

- PTSD treatment
- complex PTSD treatment
- suicide prevention outcomes
- addiction treatment outcomes
- clinical symptom reduction
- therapeutic effectiveness
- replacement of licensed care

## Pilot Flow

1. **Framework** — Define what will be measured before the pilot starts.
2. **Field Forms** — Capture attendance, feedback, fidelity, activation events, bridge activations, and unresolved handoffs.
3. **Site Review** — Aggregate operational performance and score the site.
4. **Final Report** — Summarize what happened without making clinical claims.
5. **Revision Memo** — Decide what must change before continuation or expansion.

A WRH pilot is not a performance.

A WRH pilot is a stress test.

The pilot exists to find out what works, what fails, what needs revision, and whether the site can support WRH safely before expansion.
