# WRH Pilot Operating & Evaluation Package Index

**Program Administrator:** Capitol Contracts LLC  
**Program:** What Really Happened™ (WRH)  

---

## Package Purpose

This package contains the operating instruments for a controlled WRH pilot. It is designed to measure implementation, fidelity, support-pathway performance, documentation quality, and required revision before expansion.

The package does not measure or claim clinical outcomes.

## Claims Boundary

This package supports operational conclusions only.

WRH may use this package to evaluate whether:

- the curriculum was delivered with fidelity
- facilitators stayed inside non-clinical scope
- participants engaged without required disclosure
- activation events were documented
- support bridge activations were completed or escalated
- unresolved handoffs were reviewed
- the site readiness gate accurately predicted site performance
- revision is required before expansion

WRH may not use this package alone to claim:

- clinical effectiveness
- PTSD or complex PTSD treatment
- suicide prevention outcomes
- addiction treatment outcomes
- symptom reduction
- therapeutic benefit
- replacement of licensed care

## Core Standard

**A pilot is not a performance. A pilot is a stress test.**

The purpose of this package is to make sure WRH can run safely once, measure honestly, and improve before it scales.
