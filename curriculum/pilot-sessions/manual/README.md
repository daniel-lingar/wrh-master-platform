## Archive / Draft Manual Notice

The large files named `WRH_FULL_PILOT_MANUAL_Part1.md`, `WRH_FULL_PILOT_MANUAL_Part2.md`, and `WRH_FULL_PILOT_MANUAL_Part3.md` should be treated as consolidated working drafts unless and until they are individually reviewed, split, and brought into alignment with the current public scope language.

The current WRH public scope is:

- non-clinical psychoeducation,
- no required trauma disclosure,
- no therapy-style processing,
- no diagnosis or treatment,
- facilitator-led education with host-site safety procedures,
- warm handoff when participant needs exceed WRH scope.

If any older full-manual wording conflicts with this directory notice or the cleaned modular files, the cleaned modular files and legal/compliance documents control.
