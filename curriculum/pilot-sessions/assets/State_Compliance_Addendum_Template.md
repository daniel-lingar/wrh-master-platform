# State Compliance Addendum Template

## Purpose
This document outlines the scope-of-practice review checklist for state-level implementation of the Capitol Contracts LLC trauma literacy curriculum. It is intended for organizational review to ensure compliance with state-specific statutes and crisis response protocols.

## Organizational Review Checklist

| Item | Description | Status (Y/N/NA) | Notes |
|---|---|---|---|
| **1. State Licensing Board Review** | Review by relevant state licensing boards (e.g., social work, counseling, psychology) to confirm non-clinical scope. | | |
| **2. Scope of Practice Statutes** | Verification against state statutes defining the scope of practice for peer facilitators and paraprofessionals. | | |
| **3. Crisis Response Protocols** | Integration with existing organizational crisis response and safety planning protocols. | | |
| **4. Clinical Supervision Guidelines** | Establishment of clear guidelines for clinical supervision for licensed partner organizations, where applicable. | | |
| **5. Informed Consent Procedures** | Review of informed consent procedures to ensure participants understand the non-clinical nature of the curriculum. | | |
| **6. Data Privacy & HIPAA Compliance** | Confirmation of data collection, storage, and sharing practices align with state and federal privacy regulations (e.g., HIPAA). | | |
| **7. Mandated Reporting Requirements** | Training and protocols in place for mandated reporting of abuse or neglect, as per state law. | | |
| **8. Cultural Competency Review** | Review of curriculum content and delivery methods for cultural competency and responsiveness to diverse populations within the state. | | |

## Disclaimer

⚠️ **Scope of Practice Notice:** This curriculum is designed for peer facilitators and paraprofessionals. Implementation requires organizational review of state-specific scope-of-practice statutes and crisis response protocols. Capitol Contracts LLC provides this State Compliance Addendum and clinical supervision guidelines for licensed partner organizations. Not a substitute for licensed medical or psychological treatment.
